项目简介

如何运行游戏
系统要求
​操作系统​：Windows 10 或更高版本 / macOS 10.15 或更高版本 / 主流的Linux发行版。

​存储空间​：至少需要 500 MB 的可用空间。

运行步骤
根据您的操作系统，下载对应的游戏发布包。

解压压缩包。

在解压后的目录中，找到并双击运行 grsbdj.exe（Windows）或与之类似的应用程序（macOS/Linux）。
